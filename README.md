# DownloadImagesParallel

In class assignment. This program using threading to download images from picsum.photos!


Dependencies:

Package            Version

------------------ ---------

certifi            2026.2.25

charset-normalizer 3.4.4

idna               3.11

pip                26.0.1

requests           2.32.5

urllib3            2.6.3

tqdm               4.67.3


To run the program, set up the venv first

python3 -m venv venv

source venv/bin/activate


And then change images to set the range of images you want to download.

Then run:

python3 fastFetch.py
